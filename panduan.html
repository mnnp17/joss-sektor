Pertanyaan Umum (FAQ)

Bagaimana cara mereset password akun JOSS saya?
Anda dapat melakukan reset password melalui halaman login dengan mengklik "Lupa Password". Masukkan alamat email yang terdaftar dan ikuti instruksi yang dikirimkan ke kotak masuk Anda.
Apakah sistem JOSS bisa diakses secara offline?
Tidak, sistem JOSS adalah aplikasi berbasis web yang membutuhkan koneksi internet stabil (online) untuk melakukan input data harian maupun validasi secara real-time.
Format file apa saja yang didukung untuk upload dokumen persyaratan?
Sistem JOSS saat ini mendukung dokumen berformat PDF (maksimal 5MB) untuk dokumen legalitas. Untuk gambar pendukung teknis, format JPG dan PNG diperbolehkan hingga maksimal 2MB per file.
